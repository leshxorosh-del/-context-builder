# Context Builder (Кратко на русском)

**Context Builder** — визуальный конструктор контекста для LLM, который объединяет знания из разных чатов в общую карту проекта.

## Что решает

Обычные AI-чаты фрагментируют знания. Важная информация теряется между диалогами.  
Context Builder связывает чаты в граф и позволяет формировать единый контекст для новых запросов.

## Ключевые возможности

- Визуальная карта чатов и супер-чатов
- Выбор конкретных сообщений для контекста (экономия токенов)
- Супер-чат с объединённой памятью
- LLM-дайджесты и уведомления (Telegram / Email / Slack)

## Быстрый старт

```bash
git clone https://github.com/leshxorosh-del/-context-builder.git
cd context-builder
cp .env.example .env
docker-compose up -d
```
